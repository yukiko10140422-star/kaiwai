# KAIWAI

Notion + Slack を統合した、小規模チーム向け軽量コラボレーション Web アプリケーション。

Slack 風のリアルタイムチャットと Notion 風のタスク管理を一つのアプリに統合し、既存ツールのコストを抑えつつ必要な機能を提供します。

## プロジェクト概要

- **対象ユーザー**: 小規模チーム（最大 10 名）
- **アクセス方式**: 招待制（管理者がメンバーを招待）
- **デザイン**: ダーク/ライトモード対応、グラスモーフィズム、PC ファーストのレスポンシブ設計

## 主な機能

### チャット（Slack 風）
- チャンネル・ダイレクトメッセージ
- リアルタイムメッセージング（Supabase Realtime）
- スレッド返信
- リアクション
- ファイル共有（画像・動画・ドキュメント、ドラッグ&ドロップ対応）
- @メンション
- 未読管理・通知バッジ

### タスク管理（Notion 風）
- カンバンボード（未着手 / 進行中 / レビュー / 完了）
- ドラッグ&ドロップによるステータス変更
- 複数メンバーへのアサイン
- サブタスク（チェックリスト形式）
- ラベル・タグによる分類
- 優先度設定（高 / 中 / 低）
- 期限設定・期限切れ警告
- フィルター・ソート

### プロジェクト管理
- プロジェクト CRUD
- KPI 目標設定
- 進捗トラッキング

### 議事録
- 議事録の作成・閲覧・編集・削除
- プロジェクトとの紐付け

### ダッシュボード
- 統計サマリー（完了数 / 未完了数 / 遅延数）
- 締め切りカレンダー
- メンバー別進捗表示
- 通知一覧

### 進捗ダッシュボード
- プロジェクト進捗率（円グラフ・進捗バー）
- メンバー別タスク消化状況
- 期限カレンダー

### PWA 対応
- ホーム画面への追加
- オフラインキャッシュ

### リアルタイム通知
- アプリ内トースト通知
- ブラウザプッシュ通知

### グローバル検索
- `/` キーで起動
- チャンネル・メッセージ・タスクを横断検索

### 招待制アクセス管理
- 管理者によるメール招待
- 招待リンクによるメンバー追加

## セットアップ手順

### 前提条件

- Node.js 18 以上
- npm
- Supabase アカウント

### インストール

```bash
git clone <repository-url>
cd projectcontact
npm install
```

### 環境変数の設定

プロジェクトルートに `.env.local` を作成し、以下を設定します。

```env
NEXT_PUBLIC_SUPABASE_URL=https://<your-project-id>.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=<your-anon-key>
```

Supabase プロジェクトの Settings > API から URL と anon key を取得してください。

### データベースセットアップ

Supabase の SQL Editor で `docs/schema.sql` を実行し、テーブルとポリシーを作成します。

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 にアクセスします。

### その他のコマンド

```bash
npm run build   # 本番ビルド
npm run start   # 本番サーバー起動
npm run lint    # ESLint 実行
```

## 使い方マニュアル

### ログイン・アカウント作成

1. トップページから「サインアップ」でメールアドレスとパスワードを入力して登録
2. 登録済みの場合は「サインイン」でログイン
3. 招待リンクから参加する場合は、リンクをクリックしてアカウントを作成

### ホームダッシュボード

ログイン後のホーム画面で以下を確認できます：
- タスク統計サマリー（完了数・未完了数・遅延数）
- 直近の締め切りタスク
- メンバーの進捗状況
- 通知一覧

### チャット

- **チャンネル作成**: サイドバーのチャンネル一覧から新規チャンネルを作成
- **メッセージ送信**: チャットエリア下部の入力欄からテキストを送信
- **ファイル添付**: 入力欄のクリップアイコン、またはドラッグ&ドロップでファイルをアップロード
- **メンション**: `@` を入力するとメンバー候補が表示され、選択して通知を送信
- **スレッド**: メッセージをクリックして右パネルでスレッド返信
- **リアクション**: メッセージにホバーして絵文字リアクションを追加
- **DM**: サイドバーのダイレクトメッセージ一覧からメンバーを選択

### タスク管理

- **タスク作成**: タスクボードの「+」ボタンからタイトル・説明・担当者・期限・優先度を設定
- **カンバン操作**: カードをドラッグ&ドロップしてステータスを変更
- **フィルタ**: ヘッダーのフィルターで担当者・ステータス・ラベル・優先度で絞り込み
- **詳細編集**: タスクカードをクリックしてモーダルで詳細を編集（サブタスク、コメント等）

### プロジェクト管理

- **プロジェクト作成**: プロジェクト一覧ページから新規作成
- **KPI 設定**: プロジェクト詳細画面で目標 KPI を設定し、進捗を追跡

### 議事録

- **作成**: 議事録ページから新規作成し、プロジェクトと紐付け
- **閲覧・編集・削除**: 議事録一覧から選択して操作

### 進捗確認

進捗ダッシュボードでプロジェクト全体の完了率、メンバー別の進捗、期限カレンダーを確認できます。

### 設定

- **テーマ切替**: ダーク/ライトモードを切り替え
- **プロフィール編集**: 表示名・アバター画像を変更
- **メンバー招待**: 設定画面からメールアドレスで招待リンクを送信
- **パスワード変更**: 設定画面からパスワードを更新

### キーボードショートカット

| キー | 操作 |
|------|------|
| `/` | グローバル検索を起動 |

### PWA インストール方法

- **Chrome (PC/Android)**: アドレスバー右側の「インストール」アイコンをクリック
- **iOS Safari**: 共有メニューから「ホーム画面に追加」を選択

## プロジェクト構成

```
src/
  app/                    # Next.js App Router ページ
    auth/                 # 認証（ログイン・サインアップ・招待）
    dashboard/            # メイン画面
      chat/               # チャット
      tasks/              # タスク管理
      projects/           # プロジェクト管理
      notes/              # 議事録
      progress/           # 進捗ダッシュボード
      settings/           # 設定
      profile/            # プロフィール
      dm/                 # ダイレクトメッセージ
  components/             # React コンポーネント
    ui/                   # 汎用 UI（Button, Card, Modal, Toast 等）
    layout/               # レイアウト（Sidebar, ChannelList 等）
    chat/                 # チャット関連
    tasks/                # タスク関連
    dashboard/            # ダッシュボード関連
    notifications/        # 通知関連
    search/               # グローバル検索
    settings/             # 設定関連
    pwa/                  # PWA 関連
  lib/                    # ユーティリティ・API クライアント
    supabase/             # Supabase クライアント設定
  types/                  # TypeScript 型定義
docs/                     # ドキュメント
public/                   # 静的ファイル
```
