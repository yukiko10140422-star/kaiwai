# KAIWAI - 機能要件

## 1. 認証・ユーザー管理

| 機能 | 説明 | 優先度 |
|------|------|--------|
| サインアップ | メールアドレス+パスワードで新規登録 | 必須 |
| サインイン | メールアドレス+パスワードでログイン | 必須 |
| サインアウト | ログアウト | 必須 |
| プロフィール設定 | 表示名・アイコン画像の設定・変更 | 必須 |
| パスワードリセット | メール経由でパスワード再設定 | 高 |
| 招待機能 | 管理者がメールで招待リンクを送信 | 高 |

## 2. 連絡ツール（チャット機能） - Slack風

| 機能 | 説明 | 優先度 |
|------|------|--------|
| チャンネル | トピック別のチャットルーム作成・管理 | 必須 |
| リアルタイムメッセージ | テキストメッセージ送受信（Supabase Realtime） | 必須 |
| ダイレクトメッセージ | メンバー間の1対1チャット | 必須 |
| メンション | @ユーザー名 で特定メンバーに通知 | 高 |
| ファイル共有 | 画像・動画・ドキュメント等の添付送信 | 必須 |
| メッセージ検索 | 過去のメッセージをキーワード検索 | 中 |
| 未読管理 | 未読メッセージの表示・通知バッジ | 高 |
| スレッド返信 | メッセージへのスレッド形式での返信 | 中 |
| リアクション | メッセージへの絵文字リアクション | 低 |

## 3. ファイル共有・資料管理

| 機能 | 説明 | 優先度 |
|------|------|--------|
| 画像アップロード | PNG, JPG, GIF, WebP 等 | 必須 |
| 動画アップロード | MP4, MOV 等（プレビュー再生対応） | 必須 |
| ドキュメント共有 | PDF, Word, Excel, PowerPoint 等 | 必須 |
| その他ファイル | ZIP, CSV 等あらゆるファイル形式 | 高 |
| プレビュー表示 | 画像・動画・PDFのインライン表示 | 高 |
| ドラッグ&ドロップ | ファイルをドラッグで簡単アップロード | 高 |
| ファイル一覧 | チャンネル・DM内の共有ファイル一覧表示 | 中 |
| ストレージ容量管理 | Supabase Storage の容量監視 | 中 |

## 4. タスク管理（タスク調整機能） - Notion風

| 機能 | 説明 | 優先度 |
|------|------|--------|
| タスク作成 | タイトル・説明・期限・担当者を設定 | 必須 |
| タスク一覧 | リスト表示で全タスクを確認 | 必須 |
| カンバンボード | ステータス別（未着手/進行中/レビュー/完了）のボード表示 | 必須 |
| 担当者アサイン | タスクにメンバーを割り当て（アイコン表示） | 必須 |
| 期限設定 | タスクの締切日を設定（期限切れ警告表示） | 必須 |
| ステータス変更 | ドラッグ&ドロップでステータス変更 | 必須 |
| 進捗バー | タスク全体の進捗率をビジュアル表示 | 必須 |
| ラベル・タグ | タスクの分類用カラーラベル | 高 |
| コメント | タスクへのコメント・議論 | 高 |
| フィルター・ソート | 担当者・ステータス・期限・ラベルでフィルタリング | 高 |
| プロジェクト分類 | タスクをプロジェクト単位でグループ化 | 高 |
| タイムライン表示 | ガントチャート風の期間表示 | 中 |
| サブタスク | タスク内に子タスクを作成（チェックリスト形式） | 高 |
| 優先度設定 | 高/中/低の優先度アイコン表示 | 高 |

## 5. 進捗ダッシュボード

| 機能 | 説明 | 優先度 |
|------|------|--------|
| プロジェクト進捗率 | 全タスクの完了率を円グラフ・進捗バーで表示 | 必須 |
| メンバー別進捗 | 各メンバーの担当タスク消化状況 | 高 |
| 期限カレンダー | タスク期限をカレンダー上に表示 | 高 |
| 今週のタスク | 今週期限のタスクをハイライト | 高 |
| アクティビティログ | 誰がいつ何をしたかの履歴表示 | 中 |
| 統計サマリー | 完了数/未完了数/遅延数のサマリーカード | 高 |

## 6. 通知機能

| 機能 | 説明 | 優先度 |
|------|------|--------|
| アプリ内通知 | メンション・タスクアサイン・期限アラート | 必須 |
| 通知一覧 | 通知の一覧・既読管理 | 高 |
| ブラウザ通知 | プッシュ通知（ブラウザ通知API） | 中 |
