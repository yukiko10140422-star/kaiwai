# KAIWAI - プロジェクト概要

## プロジェクト名
KAIWAI

## 目的
プロジェクト進行中のチーム内連絡・タスク調整を行うWebアプリケーション。
Notion（ドキュメント・タスク管理）+ Slack（チャット・連絡）の統合型軽量ツール。

## 背景
- 既存ツール（Notion, Slack等）はチーム利用時のコストが高い
- 必要な機能に絞った自作ツールでコストを抑える

## 対象ユーザー
- チームメンバー（最大10人程度）
- 招待制（管理者がメンバーを招待）

## 技術スタック

| カテゴリ | 技術 | 備考 |
|----------|------|------|
| フロントエンド | Next.js (React) | Vercelとの相性最適 |
| スタイリング | Tailwind CSS | モダンUI・高品質デザイン |
| アニメーション | Framer Motion | スムーズなUI遷移 |
| データベース | Supabase (PostgreSQL) | 認証・DB・ストレージ統合 |
| 認証 | Supabase Auth | メール+パスワード |
| リアルタイム通信 | Supabase Realtime | チャット・通知配信 |
| ファイルストレージ | Supabase Storage | 画像・動画・ドキュメント保存 |
| ホスティング | Vercel | Next.js最適デプロイ先 |
| 言語 | TypeScript | 型安全な開発 |
