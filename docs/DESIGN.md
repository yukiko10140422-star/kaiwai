# KAIWAI - デザイン・UI設計

## 1. デザインコンセプト
- **モダン & プレミアム**: ダークモード対応、グラスモーフィズム、微細なグラデーション
- **直感的操作**: ドラッグ&ドロップ、スムーズなアニメーション遷移
- **情報の視認性**: 進捗状況が一目でわかるビジュアルデザイン

## 2. デザインシステム
- Tailwind CSS ベースのカスタムテーマ
- Framer Motion によるページ遷移・要素アニメーション
- レスポンシブ対応（PC優先、モバイルも使用可能）
- ダークモード / ライトモード切り替え

## 3. UIコンポーネント
- グラスモーフィズムのカード（backdrop-blur + 半透明背景）
- グラデーション付きプログレスバー
- スムーズなドラッグ&ドロップ（カンバンボード）
- アバターアイコン付きメンバーリスト
- フローティング通知トースト
- ホバー時のマイクロインタラクション

## 4. 画面構成

```
1. サインイン / サインアップページ
   └── モダンな認証フォーム（グラスモーフィズム）

2. メインダッシュボード
   ├── サイドバー（折りたたみ可能）
   │   ├── ユーザープロフィール（アイコン・名前・ステータス）
   │   ├── チャンネル一覧
   │   ├── ダイレクトメッセージ一覧
   │   ├── タスクボードへのリンク
   │   └── 設定
   ├── チャットエリア（メイン表示）
   │   ├── チャンネルヘッダー（メンバー・ファイル一覧）
   │   ├── メッセージ一覧（画像・動画プレビュー対応）
   │   └── メッセージ入力欄（ファイルドラッグ&ドロップ）
   └── 右パネル（トグル表示）
       ├── スレッド詳細
       ├── メンバー情報
       └── 共有ファイル一覧

3. タスクボード
   ├── ヘッダー（プロジェクト切り替え・フィルター）
   ├── ビュー切り替え（カンバン / リスト / タイムライン）
   ├── カンバンビュー（ドラッグ&ドロップ列）
   ├── 進捗ダッシュボード（グラフ・統計）
   └── タスク詳細モーダル

4. 進捗ダッシュボード
   ├── プロジェクト全体の進捗率（円グラフ）
   ├── メンバー別タスク状況
   ├── 期限カレンダー
   └── アクティビティログ

5. プロフィール設定ページ
6. 管理者設定ページ（メンバー招待・権限管理）
```

## 5. カラーパレット（案）

### ダークモード
- 背景: `#0f172a`（slate-900）
- サイドバー: `#1e293b`（slate-800）
- カード: `rgba(255, 255, 255, 0.05)` + backdrop-blur
- アクセント: `#3b82f6`（blue-500）
- テキスト: `#f1f5f9`（slate-100）

### ライトモード
- 背景: `#f8fafc`（slate-50）
- サイドバー: `#ffffff`
- カード: `rgba(255, 255, 255, 0.8)` + backdrop-blur
- アクセント: `#2563eb`（blue-600）
- テキスト: `#1e293b`（slate-800）

### ステータスカラー
- 未着手: `#94a3b8`（slate-400）
- 進行中: `#3b82f6`（blue-500）
- レビュー: `#f59e0b`（amber-500）
- 完了: `#22c55e`（green-500）
- 期限切れ: `#ef4444`（red-500）

### 優先度カラー
- 高: `#ef4444`（red-500）
- 中: `#f59e0b`（amber-500）
- 低: `#6b7280`（gray-500）
